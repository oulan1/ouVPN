## 📖 О проекте

**ouVPN** — это проект, предоставляющий открытый доступ к надежным VPN-серверам. Наша главная цель — обеспечить свободный, анонимный и безопасный интернет для каждого пользователя. 

Все свежие конфиги, новости и обновления публикуются в нашем официальном Telegram-канале **[@oulanGift](https://t.me/oulanGift)**.

## ✨ Ключевые особенности

- 💸 **Абсолютно бесплатно** — никаких скрытых подписок и лимитов.
- ⚡ **Высокая скорость** — регулярное обновление баз серверов.
- 🔒 **Анонимность и безопасность** — обходите блокировки и защищайте свой трафик.
- 🎁 **Премиум-доступ** — специальные быстрые серверы для участников сообщества.

## 📂 Структура репозитория

В этом репозитории хранятся актуальные конфигурации и файлы проекта:

* 📄 `index.html` — стартовая веб-страница нашего проекта.
* 🌍 `servers1.txt` — актуальный список базовых VPN-серверов.
* 🚀 `premium.txt` — список премиум-серверов (обновления и пароли ищите в Telegram).

## 🚀 Как подключиться

Для использования наших серверов мы поддерживаем **только** приложение **Happ**.

1. Откройте файл [`servers1.txt`](./servers1.txt) или [`premium.txt`](./premium.txt) в этом репозитории.
2. Скопируйте нужную конфигурацию сервера.
3. Откройте приложение **Happ** и импортируйте скопированный ключ.
4. Нажмите «Подключиться» и наслаждайтесь свободным интернетом! 🌐
